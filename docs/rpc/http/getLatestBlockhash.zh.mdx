返回最新的区块哈希

<Callout type={"caution"} title={"版本限制"}>
  此方法仅在 v1.9 或更高版本中可用solana-core。v1.8 及以下版本请使用
  [getRecentBlockhash](/docs/rpc/http/getRecentBlockhash)`solana-core`。
</Callout>

<DocSideBySide>

<DocLeftSide>

### 参数

<Parameter type={"object"} optional={true}>

配置对象包含以下字段：

<Field
  name="commitment"
  type="string"
  optional={true}
  href="/docs/rpc/index.zh.mdx#配置状态承诺"
></Field>

<Field name="minContextSlot" type="number" optional={true}>
  可以评估请求的最小槽位
</Field>

</Parameter>

### 结果

`RpcResponse<object>` - RpcResponse JSON 对象，其value字段设置为 JSON 对象，包
括：

- `blockhash: <string>` - 作为 base-58 编码字符串的哈希值
- `lastValidBlockHeight: <u64>` - 区块哈希有效的最后一
  个[区块高度](/docs/terminology.md#block-height)。

</DocLeftSide>

<DocRightSide>

### 代码示例

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {
    "id":1,
    "jsonrpc":"2.0",
    "method":"getLatestBlockhash",
    "params":[
      {
        "commitment":"processed"
      }
    ]
  }
'
```

### 响应

```json
{
  "jsonrpc": "2.0",
  "result": {
    "context": {
      "slot": 2792
    },
    "value": {
      "blockhash": "EkSnNWid2cvwEVnVx9aBqawnmiCNiDgp3gUdkDPTKN1N",
      "lastValidBlockHeight": 3090
    }
  },
  "id": 1
}
```

</DocRightSide>
