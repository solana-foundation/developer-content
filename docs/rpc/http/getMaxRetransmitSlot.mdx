Get the max slot seen from retransmit stage.

<DocSideBySide>

<DocLeftSide>

### Parameters

**None**

### Result

`<u64>` - Slot number

</DocLeftSide>

<DocRightSide>

### Code sample

```shell
curl https://api.devnet.solana.com -X POST -H "Content-Type: application/json" -d '
  {"jsonrpc":"2.0","id":1, "method":"getMaxRetransmitSlot"}
'
```

### Response

```json
{ "jsonrpc": "2.0", "result": 1234, "id": 1 }
```
