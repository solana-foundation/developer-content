Returns the slot leaders for a given slot range

<DocSideBySide>

<DocLeftSide>

### Parameters

<Parameter type={"u64"} optional={true}>
  Start slot, as u64 integer
</Parameter>

<Parameter type={"u64"} optional={true}>
  Limit, as u64 integer (between 1 and 5,000)
</Parameter>

### Result

`<array[string]>` - array of Node identity public keys as base-58 encoded
strings

</DocLeftSide>

<DocRightSide>

### Code sample

If the current slot is `#99` - query the next `10` leaders with the following
request:

```shell
curl https://api.devnet.solana.com -s -X POST -H "Content-Type: application/json" -d '
  {
    "jsonrpc":"2.0", "id": 1,
    "method": "getSlotLeaders",
    "params": [100, 10]
  }
'
```
